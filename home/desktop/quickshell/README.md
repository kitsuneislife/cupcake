QuickShell example configuration

This directory contains example QML and JSON files for QuickShell.
- `main.qml` — minimal placeholder top‑bar UI
- `panel.qml` — panel fragment used by `main.qml`
- `settings.json` — example runtime settings (autostart, position)

To use your own configuration, replace these files or mount a different folder at `~/.config/quickshell/`.
